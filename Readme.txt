A ideia é criar um chat no terminal. Usando o Console project do .NET em C#, quero que usuários possam enviar mensagens que serão compartilhadas em tempo real entre todos os clientes abertos.
Para isso, precisarei fazer uma espécie de WebSocket CLI, para o envio e recebimento de mensagens, além de uma forma de armazená-las. Dado minha familiaridade, estou usando SQLite.
Para criar a interface estou usando o Console.SetPositionCursor, é a base para montar a interface e exibir as mensagens.
O projeto não está pronto, por favor, dê uma olhada: https://github.com/Andersignalr/Rectangle-Drawer-and-Messages
